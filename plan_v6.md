## 2. 提案1+2 が生む構造上の根本変更:**「上部固定 / 下部回転」へ転換**

### 現在(V5)
すべての部材が `rotatingGroup` の中で一緒に回転している(中心軸だけは独立して立つだけ)。L3十字も屋根も高支柱もキャスターで床を転がっている前提。

### 提案後(V6)
- **中心支柱** は床合板に金具で**固定**(回らない)
- **中心支柱の上端**は L3 X-pipe と L3 Z-pipe にクランプで**固定**
- → 中心支柱とL3十字が**一体の静止フレーム**になる
- → 屋根もそれに乗っているので**静止**
- → 高支柱4本(端、r=2.05)も静止フレームの一部とすれば屋根を支える脚として機能

これは**実機の一般的なメリーゴーランド構造**(中心柱は固定、下のプラットフォームが回る)に近づく。

**回転する部材** : L1井形 / L2デッキ(再設計) / 馬ポール4本 / 木の支柱4本(対角)
**静止する部材** : 中心支柱 / L3十字 / 屋根 / 高支柱4本(端) / 波形レール / 床合板 + 重し

## 5. 確認したい設計上の判断

下記のいくつかは複数の解釈があるので、進める前に確認したい点:

### Q1. 高支柱(端、r=2.05)の扱い
- **(a)** 静止フレームとして固定(屋根を支える脚にする) — 推奨
- (b) 撤去(屋根は中心支柱のみで支える)
- (c) 回転させたまま(屋根とは別物として、回転プラットフォームの外側ガイドとして)

### Q2. L2 補強板の配置
- **(a)** 馬ポール周りに 4 枚のみ(推奨) — 上記「案A」
- (b) 各パイプに沿って多数(案B/C)

### Q3. 波形レールの DIY 構造可視化レベル
- **(a)** 基台リング追加 + 色変更のみ(簡易) — 推奨
- (b) 紙束ウェッジを周方向に多数配置(詳細)

### Q4. 重しの形状とサイズ
- (a) 円柱状の鉄重し(2-5kg ダンベル風)を 4 個 — 推奨
- (b) 砂袋風(布製、立方体に近い)複数個

### Q5. 木の支柱(対角、r=1.65)の扱い
- 現状は回転側に残す
- L2 デッキの大半がベニヤ(軽量)になるなら、木の支柱の本数や太さも見直す価値あり

## 6. 物理的観点(再検証ポイント)

### 6.1 中心支柱固定式に変えた場合の力学
- 床合板+重し合計 ~30 kg で中心支柱の転倒モーメントを抑える
- L3 と中心支柱の連結で、屋根+L3+高支柱の重量(75kg程度)が中心支柱に分配
- 中心支柱(単管 φ48.6mm × 2.6m × 4kg = 約10kg, 引張耐力20kN)で安全率十分

### 6.2 回転プラットフォーム単体の挙動
- 静止フレームと分離されたことで、回転側の重量は L1+L2新+馬ポール+木の支柱 ≈ 250kg(ベニヤ化で大幅軽量)
- 木の支柱4本(キャスター付)で支持、回転中の重心が安定

### 6.3 摩擦による手押し力
- V5 算定 25kgf → V6 では合板軽量化で **約15 kgf** に低減見込み(2 人で楽に運用可)

### 6.4 残課題
- レール内側(r < 0.84)に床合板を敷くのは OK だが、レール外周(r > 1.16)の空間の処理 → 回転プラットフォームが床上を浮く形になるが、これは木の支柱のキャスターが受ける ✓
